# プロジェクト固有の知識ファイル置き場（プロジェクト層）

プロジェクト固有の知識（言語規約・ドメイン知識・運用手順など）はこのディレクトリに置き、
`.aidd-docs/project/context.md` の「プロジェクト固有の知識ファイル」表に**読むタイミングとともに登録する**。

- ここに置いたファイルは**該当する作業のときだけ**読み込まれる（毎セッション読み込まれるのは context.md のみ）。context.md を薄い索引に保つため、分量のある内容は必ずこちらへ
- テンプレート側の `.aidd-docs/knowledge/` には追加しない（テンプレート更新で上書きされる領域のため）
- 書式はテンプレートの知識ファイルと同じでよい（frontmatter の `name` / `description` + チェックリスト・アンチパターン集）
- Claude Code でスキルとして自動発見させたい場合は、`.claude/skills/` のテンプレート由来ローダーと同様に、ここを参照する薄いローダーを追加してよい（本文はこちら側に置く）
